# CCIU

CCI 结合趋势战略，其中CCI是原值*0.13+50，超过90即为超买，低于10为超卖

#### code

```
VARA:=LLV(LOW,35);
VARB:=HHV(HIGH,30);
VARC:=EMA((CLOSE-VARA)/(VARB-VARA)*4,4)*25;
趋势: VARC,COLORFF00FF,LINETHICK2;
底背: STICKLINE(VARC<10 AND 100*(HHV(HIGH,14)-CLOSE)/(HHV(HIGH,14)-LLV(LOW,14))>97,0,25,5,0),coloryellow;
顶背: STICKLINE(VARC>90,100,85,10,0),colorred;
VARD:=87.5;
VARE:=(VARC-LLV(VARC,4))/(HHV(VARC,4)-LLV(VARC,4))*4*25;
目标位: DRAWTEXT(IF(CROSS(趋势,VARD) AND VARE=100,50,0),66,'目标'),COLORFF00FF;
跑: DRAWTEXT(IF(CROSS(VARD,趋势),87,0),90,'跑'),COLOR00FF00;
卖: DRAWTEXT(IF(CROSS(趋势,VARE) AND 趋势>87,87,0),78,'卖'),COLOR00FF00;
买: DRAWTEXT(IF(CROSS(趋势,20) AND 趋势>20,20,0),15,'买'),COLORFF0000;
VAR6:=(2*C+H+L)/4;
VAR8:=LLV(LOW,30);
VARA1:=HHV(HIGH,30);
B:=EMA((VAR6-VAR8)/(VARA1-VAR8)*100,8);
B1:=EMA(B,5);
STICKLINE(B>0 AND B-B1>=0,79,80,6,0),COLOR00FF00;
STICKLINE(B>0 AND B-B1<0,79,80,6,0),colorcyan;
STICKLINE(B>0 AND B-B1>=0,49,50,6,0),colorred;
STICKLINE(B>0 AND B-B1<0,49,50,6,0),colorcyan;
STICKLINE(B>0 AND B-B1>=0,19,20,6,0),COLOR00FF00;
STICKLINE(B>0 AND B-B1<0,19,20,6,0),colorcyan;
STICKLINE(B-B1>0,B,B1,8,1),colorred;
STICKLINE(B-B1<0,B,B1,8,1),COLOR00FF00;
V1:=(2*CLOSE+HIGH+LOW)/4;
V2:=LLV(LOW,21);
VA:=HHV(HIGH,34);
防线B:=EMA((V1-V2)/(VA-V2)*100,13);
防线BB:=EMA((0.667*REF(防线B,1)+0.333*防线B),2);
战略:EMA(0.382*REF(防线B,2)+0.618*防线B,12),coloryellow;

TYP := (IF(ISNULL(HIGH),CLOSE,HIGH) + IF(ISNULL(LOW),CLOSE,LOW) + CLOSE)/3;
CCI:(TYP-MA(TYP,24))/(0.015*AVEDEV(TYP,24))*0.15 + 50,colorligray;
HORLINE(1,65,2,1);
HORLINE(1,35,2,1);
```

# KDJU

KDJ 结合主力吸货，金叉买入，死叉卖出

#### code

```
RSV=(CLOSE-LLV(LOW,N))/(HHV(HIGH,N)-LLV(LOW,N))*100;
a=SMA(RSV,M1,1);
b=SMA(a,M2,1);
e=3*a-2*b;
K:a;
D:b;
J:e;

VAR1:=REF((LOW+OPEN+CLOSE+HIGH)/4,1);
VAR2:=SMA(ABS(LOW-VAR1),13,1)/SMA(MAX(LOW-VAR1,0),10,1);
VAR3:=EMA(VAR2,10);
VAR4:=LLV(LOW,33);
VAR5:=EMA(IF(LOW<=VAR4,VAR3,0),3);
Z:0;
进场:IF(VAR5>REF(VAR5,1),VAR5,0),colorred,NODRAW;
STICKLINE(VAR5>REF(VAR5,1),0,VAR5,8,0),colorred;
洗盘:IF(VAR5<REF(VAR5,1),VAR5,0),colorgreen,NODRAW;
STICKLINE(VAR5<REF(VAR5,1),0,VAR5,8,0),colorgreen;
```

# DDEU

DDE大单净量，金叉买入，死叉卖出


#### code

```
LC:=REF(CLOSE,1);
AA:=ABS(HIGH-LC);
BB:=ABS(LOW-LC);
CC:=ABS(HIGH-REF(LOW,1));
DD:=ABS(LC-REF(OPEN,1));
R:=IF(AA>BB AND AA>CC,AA+BB/2+DD/4,IF(BB>CC AND BB>AA,BB+AA/2+DD/4,CC+DD/4));
X:=(CLOSE-LC+(CLOSE-OPEN)/2+LC-REF(OPEN,1));
r:=16*X/R*MAX(AA,BB);
ASI1:=SUM(r,0);
STICKLINE(r>0,0,r*2.5,9,0),color0000ee;
STICKLINE(r<0,0,r*2.5,8,0,),colorgreen;
DDX1:SMA(MA(r,1,1),2,1)*5,colorwhite;
DDX8:=MA(DDX1,3);
DDX2:SMA(DDX8,7,2),color0000FF,linethick2;
DDX3:EMA((DDX1+DDX2)/2,2),color00EE00;
轴:0,color664444,linethick1;
```

# WRU

WR 结合 CCI 判断趋势, 大于80超买 小于20超卖

#### code

```
TYP := (IF(ISNULL(HIGH),CLOSE,HIGH) + IF(ISNULL(LOW),CLOSE,LOW) + CLOSE)/3;
CCI:(TYP-MA(TYP,24))/(0.015*AVEDEV(TYP,24))*0.15 + 50,colorligray;

WR:100*(HHV(HIGH,42)-CLOSE)/(HHV(HIGH,42)-LLV(LOW,42));
P:(CCI-WR)/2 + 50;
HORLINE(1,80,1,2);
HORLINE(1,20,1,2);
```
